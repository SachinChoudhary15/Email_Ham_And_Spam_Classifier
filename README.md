
# 📧 Email Ham and Spam Classifier

## 📌 Project Overview

This project is a Machine Learning based Email Ham and Spam Classifier that predicts whether an email message is **Spam** or **Ham (Not Spam)**. The model is trained using text preprocessing techniques and a supervised machine learning algorithm.

---

## 🚀 Features

* Email Spam Detection
* Text Preprocessing
* Feature Extraction using TF-IDF Vectorizer
* Machine Learning Classification
* Fast and Accurate Predictions
* Easy to Use

---

## 🛠️ Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* Joblib

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/SachinChoudhary15/Email_Ham_And_Spam_Classifier.git
```

Go to the project folder

```bash
cd Email_Ham_And_Spam_Classifier
```

Create a virtual environment

```bash
python -m venv EmailClassifier
```

Activate the virtual environment

### Windows

```bash
EmailClassifier\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python model.py
```
